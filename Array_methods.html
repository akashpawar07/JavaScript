Array Methods

  1. pop() --method is used for delete a item from last position of array

  2. push() --method add's a new item to the end of an array

  3. shift() --method is used for remove 1st element of an array

  4. unshift() --method is used for add new element at the starting of an array

  5. slice() --used to create a copy of portion of an array into a new array

  6. splice() --it changes the content of an array by removing ro repalcing existing elements and/or adding new elements in place.

  7. indexOf() --method is searches for the specified object in a range of elements of a one-dementional array, and returns the index of its first occurance.

  8. lastIndexOf() --method is used to search the postion of a particular element in a given array
